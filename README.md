# 火柴人冒险 (StickMan Adventure)

一个类似《卡皮巴拉GO》的竖屏冒险游戏，玩家扮演火柴人主角，通过随机事件和战斗来成长。

## 游戏特色

- **竖屏设计**: 适配手机和电脑屏幕
- **随机事件系统**: 多种随机事件影响角色属性
- **战斗系统**: 回合制战斗，策略性强
- **角色成长**: 通过事件和战斗提升属性
- **精美UI**: 现代化的界面设计

## 游戏玩法

### 基础属性
- **生命值**: 角色的生存能力，初始100
- **攻击力**: 影响战斗伤害，初始1
- **攻击速度**: 影响战斗节奏，初始1
- **免伤**: 减少受到的伤害，初始0

### 游戏流程
1. 点击"开始冒险"开始游戏
2. 遭遇随机事件，做出选择影响属性
3. 每隔3-7个事件会触发战斗
4. 战斗中可以选择攻击、防御或逃跑
5. 通过战斗获得经验值和奖励
6. 生命值归零时游戏结束

### 随机事件
- 神秘宝箱
- 商人药水
- 受伤的旅行者
- 道路选择
- 古老祭坛
- 野狼遭遇
- 废弃营地
- 神秘谜题

### 战斗系统
- **攻击**: 对敌人造成伤害
- **防御**: 减少受到的伤害
- **逃跑**: 有概率逃离战斗

### 敌人类型
- 哥布林
- 骷髅战士
- 野狼
- 强盗
- 暗影刺客
- 石像鬼

## 技术实现

### 前端技术
- **HTML5**: 游戏界面结构
- **CSS3**: 响应式设计和动画效果
- **JavaScript**: 游戏逻辑和交互
- **Canvas**: 角色绘制和动画

### 项目结构
```
StickManAdventure/
├── index.html          # 主页面
├── styles.css          # 样式文件
├── game.js            # 游戏逻辑
└── README.md          # 项目说明
```

## 如何运行

1. 克隆或下载项目文件
2. 在浏览器中打开 `index.html`
3. 开始游戏！
